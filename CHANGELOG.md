# Changelog

## v1.0.0 (2026-03-22)

### Features
- **ARIMA** forecasting with automatic order selection (p,d,q)
- **Prophet-style** exponential smoothing with trend, seasonality, and holiday decomposition
- **Neural network** forecasting with configurable feedforward architecture
- **Changepoint detection** with BIC/AIC penalty-based structural break identification
- **Seasonal-trend decomposition** (STL-style) with configurable period
- **Feature engineering**: lag features, rolling statistics, Fourier terms
- **Forecast metrics**: MAE, RMSE, MAPE, MASE, R-squared
- **PyQt6 GUI**: forecast, decompose, changepoint detection pages with charts
- **CLI**: forecast, decompose, changepoints, features commands
